# 🍗 Nas's Fried Chicken (NFC) - Frontend App

## Project Description
This is the frontend user interface for Nas's Fried Chicken. It is a responsive, interactive web application that allows users to view the menu, search for specific food items, filter by category, and place orders seamlessly. 

## Features Implemented
* **Dynamic Data Rendering:** Fetches and displays menu data from the connected Node.js backend.
* **Advanced Feature - Search:** A search bar allowing users to find specific menu items instantly.
* **Advanced Feature - Filtering:** Category buttons that filter the menu view (All, Spicy, Original, Sides).
* **Interactive Checkout:** A functioning cart system that calculates the total price and submits the order to the database.
* **UX Handling:** Includes visual loading states (e.g., "Processing..." buttons) and error handling to keep the user informed during network requests.

## Tech Stack
* **Languages:** HTML5, CSS3, Vanilla JavaScript
* **API Communication:** Fetch API
* **Deployment:** Vercel

## Setup Instructions & Running Locally
1. Clone this repository to your local machine.
2. Ensure the backend server is running locally on port 3000.
3. You can run this project simply by opening the `index.html` file in any modern web browser, or by using a local development server like VS Code's "Live Server" extension.
4. Note: If running locally, ensure the fetch URLs in `app.js` point to your local backend (e.g., `http://localhost:3000/api/menu`).
